# Brace_exploration_robot
Program of the robot produced using Raspberry Pi

## Specification (ver2.0)
* Raspberry Pi B+
* Raspberry Pi CAMERA MODULE V1
* NiMH battery 6V 4200mAh 25.2Wh
* NiMH battery 1.2V 1900mAh (eneloop) x4
* Servomotor x4 (ASV-15-A x2, Large x2)
* Power LED (Edison EDEW-1LA5 1W White)
* Wireless LAN　(BUFFALO WLI-UC-GNM2)
* TAMIYA 楽しい工作シリーズ 
  * No.100 トラック＆ホイールセット
  * No.142 ラダーチェーン＆スプロケットセット
  * No.143 ユニバーサルアームセット
  * No.172 ユニバーサルプレート
